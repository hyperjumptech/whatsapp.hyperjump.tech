## Prerequisites

- Node.js 20+
- pnpm
- Docker

## Installation

From the root of the monorepo project, run the following command:

```bash
pnpm install
```

## Development

- Run `docker-compose up` to start the database.
- Run `pnpm run dev --filter web` to start the development server.
- Run `pnpm run build` to build the app.
- Run `pnpm run test` to run the tests.
- Run `pnpm run coverage` to run the tests with coverage.
